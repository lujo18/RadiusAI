class BrandSettings(BaseModel):
    """Account-level brand settings for AI content generation"""
    niche: str
    aesthetic: str
    target_audience: str
    brand_voice: str
    content_pillars: List[str]

    # EXISTING OPTIONAL FIELDS
    tone_of_voice: Optional[Literal["professional", "casual", "humorous", "edgy", "inspirational"]] = None
    emoji_usage: Optional[Literal["none", "minimal", "moderate", "heavy"]] = None
    forbidden_words: Optional[List[str]] = None
    preferred_words: Optional[List[str]] = None
    hashtag_style: Optional[Literal["niche", "trending", "mixed"]] = None
    hashtag_count: Optional[int] = None
    hashtags: Optional[List[str]] = None

    # NEW RECOMMENDED FIELDS FOR CAROUSEL/SLIDESHOW TEMPLATES
    
    # Format Preference
    preferred_formats: Optional[List[str]] = None  # ["transformation", "educational", "myth-busting", etc.]
    format_mix_strategy: Optional[Literal["educational_focus", "entertainment_focus", "balanced"]] = None
    primary_content_funnel_stage: Optional[Literal["awareness", "consideration", "decision", "loyalty"]] = None
    
    # Slide Design & Text
    visual_style: Optional[Literal["minimalist", "colorful", "data_heavy", "lifestyle", "product_focused"]] = None
    text_alignment: Optional[Literal["left", "center", "mixed"]] = None
    design_system_colors: Optional[List[str]] = None  # e.g., ["#FF6B6B", "#4ECDC4"]
    font_hierarchy_preference: Optional[Literal["bold_headers", "balanced", "subtle"]] = None
    
    # Hook & Opening Strategy
    hook_style_preference: Optional[List[str]] = None  # ["vulnerability", "curiosity", "problem", "transformation"]
    problem_statement_type: Optional[Literal["pain_point", "aspiration", "fear", "opportunity"]] = None
    audience_archetype: Optional[str] = None  # e.g., "ambitious founder", "fitness enthusiast", "career-changer"
    
    # Content Specificity
    transformation_claim: Optional[str] = None  # e.g., "fitness", "business", "mindset", "lifestyle"

## KEY INSIGHTS FOR CONTENT GENERATION

**When prompting the AI with these templates:**

1. **Always include the specific `template_id`** — this locks in the structure consistency
2. **Map BrandSettings to template** — if `tone_of_voice="edgy"`, route to myth-busting or pattern-interrupt formats
3. **Validate data availability** — authority block and expert formats require research; transformation format requires real before/after
4. **Mention the engagement benchmark** — AI will optimize copy toward "42% swipe-through" rather than generic "good engagement"
5. **Specify platform in prompt** — flipbook is Instagram-only; expert authority works best on LinkedIn
6. **Include audience archetype** — "ambitious founder" vs "fitness enthusiast" changes hook language entirely
7. **Flag any brand constraints** — if brand avoids vulnerability, skip narrative bridge template
8. **Test completion rates** — completion rate is THE metric for carousel success; other metrics follow from that
